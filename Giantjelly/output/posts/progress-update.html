Progress update

Good day!

Sorry we haven't said anything in a looong time. I didn't want to write a post until we had gotten a reasonable amount of progress. We've been quite busy with work so we haven't done as much as I would have liked but here's what we've got so far:

  • Added ships for the second faction "The Confederacy"
  • Ship weapons can be changed at space stations
  • Added weapon types, rockets and homing missiles
  • Particle effects
  • Explosion animations
  • Health and shield level bars
  • Collecting dropped cargo
  • Enemy AI now follow a route and stop to fight when enemies are nearby
  • Ship energy shield damage effect

Nathan has been working on some nice sound effects and music so we should have some of that soon. I also re-wrote all the code in C/C++ at Christmas which is another reason it's taken a while to get to this point. But I'm very happy with the architecture now so progress should be a bit more consistent and I'll try to give you more regular updates.

Thanks!

--Matt
