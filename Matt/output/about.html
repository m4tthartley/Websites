About

Hi, I'm Matt, a computer programmer and game developer living in rainy England.

I love games and there have been many games that I've loved over the years but the ones that stand out are The Elder Scrolls IV: Oblivion, which is probably my favorite game and The Last of Us.

Programming and Game dev

I have used many different programming and game dev tools in the past, including Java, Lua and Unity3D. I could never quite find my home but eventually I found that I really like programming in C/C++. And then Handmade Hero came along and showed how fun it is to program from scratch at a lower level. Ever since I started watching Handmade Hero I have programmed the Handmade way.

Thank you Casey Muratori for making programming fun again!

At the moment I'm working on a game in C/C++ using OpenGL for GPU rendering and no libraries except the STB libraries cause they're pretty awesome.

Check out my games at giantjelly.net
